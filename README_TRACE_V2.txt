Prasia Guild Trace - Trace v2

적용 규칙
1. 92+ 관측 인원이 3명 이상이면 기존 92+ 기준 유지
2. 92+ 관측 인원이 1~2명이면 91레벨을 보조 지문에 포함
3. 92+ 관측 인원이 0명이면 실제 존재하는 상위 3개 레벨 사용
4. 전체 관측 members는 보존하고, 비교 대상은 match_members에 별도 저장
5. 결사 순위와 점수는 data/Who_are_you_guild_score.json 기준
6. 웹 기본 이전 결사 범위는 스코어 순위 1~50위, 이후 후보는 1~500위
7. YML 경로/의존성 변경 없음
